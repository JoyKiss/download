慕课网（www.imooc.com）视频批量下载工具

**慕课网视频批量下载工具 v1.2**

**说明**：
该工具是Java写的，时间紧促，懒得写界面了，这么点交互也用不着去写界面。Java嘛，Java的运行环境是必须的，如果没有安装，可以先去下载安装：[Java运行环境下载](http://www.java.com/zh_CN/download/)。如果懒得安装，这里也提供不用自己安装Java运行环境的包（`64位`的，32位系统的还是自己下载Java运行环境吧），下载带jre的那个版本即可，但体积比较庞大。

解压缩后运行`run.bat`，根据提示操作即可。视频下载到当前工具所在目录`download`文件夹下。若需要同时下载多个课程，运行多个`run.bat`即可。

**功能**：
- 下载慕课网视频~~可选普清、高清、超清进行下载。
- 下载视频附件。课程若有附件会自动下载到课程目录下的`课程资料`文件夹。
- 课程信息预览。自动生成课程信息预览到`course_info.txt`，可以了解课程介绍、课程须知等信息。
- 课程列表。自动生成课程所有小节的慕课网在线学习的原始地址列表`course_list.html`，区分视频和非视频小节，若课程有非视频小节，将以`*`号作为标记，待有网环境下继续学习。
